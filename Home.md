[HyperApp](https://github.com/hyperapp/hyperapp) is a JavaScript library for building frontend applications. In this way, it's similar to others like React, Vue.js and Mithril.

```jsx
app({
    model: "Hi.",
    view: model => <h1>{model}</h1>
})
```
